# Calculadora em Java

## Sobre o Projeto

Este projeto é uma calculadora desenvolvida em Java que permite realizar operações matemáticas básicas através de um menu interativo no terminal.

### Funcionalidades

* Soma
* Subtração
* Multiplicação
* Divisão
* Potenciação
* Menu interativo
* Encerramento da aplicação pelo usuário

## Tecnologias Utilizadas

* Java
* Scanner (`java.util.Scanner`)
* Math (`Math.pow()`)

## Como Executar

1. Compile o programa:

```bash
javac calculadora.java
```

2. Execute o programa:

```bash
java calculadora
```

## Exemplo de Uso

```text
1 - Soma
2 - Subtração
3 - Multiplicação
4 - Divisão
5 - Potência
0 - Sair

Escolha uma opção: 1

Insira o número 1: 10
Insira o número 2: 5

10.0 + 5.0 = 15.0
```

## Conceitos Praticados

* Estruturas de repetição (`while`)
* Estruturas de decisão (`switch`)
* Entrada de dados com `Scanner`
* Operações matemáticas em Java
* Manipulação de variáveis
